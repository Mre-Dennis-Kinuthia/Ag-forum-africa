# Ag Forum Africa

A continental forum connecting farmers, agribusiness leaders, technologists, and policymakers to exchange knowledge and collaborate on agricultural innovation across Africa.

## Overview

Ag Forum Africa brings together a pan-African community around agriculture through:

- **Forums** — Discussions on crop management, livestock, agritech, agribusiness, climate, and policy
- **Knowledge Hub** — Guides, videos, research, and resources
- **Events** — Webinars, roundtables, and virtual conferences
- **Success Stories** — Impact stories from the community
- **Community** — Member profiles and networking

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run start` — Start production server
- `npm run lint` — Run ESLint
